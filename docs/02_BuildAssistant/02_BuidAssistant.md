# Exercise 02: Build the Zava Order and Returns assistant with Copilot Studio and Power Platform

## Exercise learning objectives:

- Create a Copilot Studio Agent that Zava customers can use to check order status, track deliveries, start returns/exchanges, and ask product or support questions.
- Keep the agent in the same Managed Dev environment, using solution packaging, managed environments, and Git for ALM.
