# Exercise 01: Modernize the service desk with Power Apps and Managed Platforms

## Exercise learning objectives:

- Build a modern solution and user experiences to automate, manage, and monitor order processing by integrating with enterprise ERP and CRM systems and replacing legacy systems for customer enquiries.  
- Start in a Managed Dev environment and package everything in a solution as your "source code".  

## Duration
Estimated exercise time (minutes): **90 minutes**
